# academy-site

Standalone, zero-build academic projects page using vanilla HTML, CSS, and JavaScript.

## Edit content

All page content is managed in `/Users/luisrondow/dev/academic.rondow.dev/content.js`.

You can edit:

- `meta`: title and description
- `hero`: heading, intro, and top links
- `ascii.art`: terminal-style ASCII block
- `projects`: manual order, descriptions, tags, and links
